# 扩展说明

扩展说明

为了平衡工具的复杂度，更加灵活的安装、配置、扩展工具的功能，CYB放弃了高度集成的封闭模式，交予项目自行安装、配置第三方工具库、及应用框架。如果使用cyb init命令创建项目，大部分的应用场景会自动配置，此处的帮助说明主要用于帮助开发者改造旧应用，或扩展已创建应用。
