# 部署自定义文件

# 文件类型

  • 音频文件(mp3/WAV/Mid等)
  • 视频文件（Mp4/MOV/AVI等）
  • 图像文件(jpg/jpeg/png/svg等)
  • Flash(swf)文件
  • 文档格式的文件（Doc/PPT/PDF等）
  • 无需编译的样式或脚本文件

# 部署目录

project
└── src                     
    └── custom
         ├── 文件...
         ├── 文件...                   
         └── folder           # 可以自定义创建目录（可选）
              ├── 文件...                   
              └── 文件...

发布项目时会自动将custom目录中所有文件不做任何编译，不添加版本号，并发布到dist目录。dist/custom/*
