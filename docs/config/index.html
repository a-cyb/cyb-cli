# 开发及配置说明

配置说明

塞伯坦CYB前端模块化工程构建工具使用了零配置、开箱即用的理念，根据前端开发最佳实践CYB已默认开启了大部分配置选项，包括智能提取公共模块、合并压缩、添加版本号等流程。每个项目根据实际需求可以进行更加详细、个性化的配置。
